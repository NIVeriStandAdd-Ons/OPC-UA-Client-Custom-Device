OPC Connection and Tag Status

The OPC Connection and Tag Status section contains channels that provide basic status information back to the user on the tag and OPC connection status.

The channels are:

Total Tags: Contains the total number of OPC tags being read and written.

Bad Tags: Contains the total number of OPC tags that have a status other than Good.

Good Tags: Contains the total number of OPC tags that have a Good status.

Poll Engine Error Code: Returns an error code if there is a problem with the OPC connection.
