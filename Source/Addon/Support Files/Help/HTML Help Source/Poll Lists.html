OPC Poll Lists

OPC Poll Lists provides the ability to set different poll rates for different groups of channels.

What do you want to do?

Set Default Poll Rate

To set the default poll rate, update the Default Poll Rate (ms) to the desired setting. This setting will apply to all channels NOT in another poll list. E.g. If there are no additional poll lists, all channels will be read at the default rate.

Add New Poll List

To add a new poll list, select the Add Poll List button at the top of the page. Do this as many times as you want additional poll rates. To set the new poll rate and to select the affected channels, go to the new poll rate page.

Delete multiple items

To delete multiple items, right-click the object in the configuration tree and select Delete Multiple Items from the shortcut menu.
